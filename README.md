# MyTNH Intranet

Internal portal for The Nuclear House.

## Setup

1. Clone this repo
2. Connect to Vercel
3. Done

## Adding your hero image

Drop your AI-generated control room image into `public/hero-image.jpg`

## Updating links

Edit the `primaryApps` and `quickLinks` arrays in `src/components/MyTNHPortal.jsx` to point to your actual URLs.

## Local development (optional)

```bash
npm install
npm run dev
```
